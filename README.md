# 🚀 Landing Pages Responsivas (HTML, CSS e Bootstrap Grid)

## 📌 Descrição da Atividade

Este projeto foi desenvolvido como parte da disciplina **Desenvolvimento Web (IFCE - Campus Aracati)**, ministrada pelo professor **Silas Santiago**.
O objetivo é **criar duas Landing Pages responsivas**, aplicando conceitos de **HTML, CSS e o sistema de Grid do Bootstrap (BS5)** para treinar boas práticas de design de interfaces web.

Uma **Landing Page (ou página de captura)** é uma página independente que incentiva o usuário a realizar uma ação (Call to Action - CTA), como *inscrever-se*, *comprar agora* ou *entrar em contato*.

---

## 🎯 Requisitos

* **HTML e CSS:**

  * Uso de seletores (tag, id, classe, descendente).
  * Pseudoclasses (ex.: `hover`).
  * Background images.
  * Customização de cores, bordas, padding e margens.

* **Bootstrap 5 (Grid System):**

  * Criação de interfaces responsivas para **desktop, tablet e mobile**.
  * Hierarquia visual clara (títulos, textos e botões).

* **Funcionalidades obrigatórias:**

  * Duas Landing Pages em pastas diferentes.
  * Um formulário de contato (na própria página ou em uma segunda página).
  * Estrutura organizada de pastas (`/css`, `/img`, `/js`, etc).
